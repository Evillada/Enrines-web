# Instrucciones para las imágenes

Para que el sitio web funcione correctamente, se debe añadir las siguientes imágenes en formato JPG,PNG a esta carpeta:

1. `logo.jpg` - Logo de Enrines ICOR (usado en el encabezado y pie de página)
2. `hero.jpg` - Imagen principal para la sección hero (foto de la fachada)
3. `reparacion.jpg` - Imagen para el servicio de reparación de rines
4. `venta.jpg` - Imagen para el servicio de venta de rines y llantas
5. `alineacion.jpg` - Imagen para el servicio de alineación y balanceo
6. `pintura.jpg` - Imagen para el servicio de pintura y personalización
7. `gallery1.jpg` - Primera imagen de la galería (vehículo con rines)
8. `gallery2.jpg` - Segunda imagen de la galería (vehículo con rines)
9. `gallery3.jpg` - Tercera imagen de la galería (vehículo con rines)
10. `gallery4.jpg` - Cuarta imagen de la galería (vehículo con rines)
11. `gallery5.jpg` - Quinta imagen de la galería (vehículo con rines)
12. `gallery6.jpg` - Sexta imagen de la galería (vehículo con rines)

## Recomendaciones:

- Asegúrase de que las imágenes sean de alta calidad pero optimizadas para web
- Para el logo, utiliza una imagen con fondo transparente si es posible
- Las imágenes de la galería deben tener todas la misma relación de aspecto (16:9 recomendado)
- Para las imágenes de servicios, usa fotos que muestren claramente cada tipo de servicio
